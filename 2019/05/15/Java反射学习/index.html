Java反射学习

Posted on 2019-05-15

概念

运行时的类信息

有什么用

有时程序无法在编译期就知道某个对象属于哪个类，需要在运行时确定这个对象属于哪个类，获取类的属性和方法等信息，并对它进行操作。

示例

一个把Bean数组转换为Map数组的示例，程序在编译时并不知道要转化的Bean的具体类型，使用反射来实现获取运行时的类信息。

输出结果：

[{price=, name=大嫖客2, description=天下第一, type=, class=class com.wisesoft.reflectdemo.Game}]
